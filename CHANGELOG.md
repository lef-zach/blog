# Changelog

## [1.0.0] - 2026-01-30

### Initial Release
*   Full-stack blogging platform with Next.js and Express.
*   **Auth**: Secure HttpOnly cookie-based authentication with reuse detection.
*   **Core**: Article management, Rich Text Editor, Google Scholar integration.
*   **Security**: Helmet headers, Rate Limiting, input sanitization.
*   **Infra**: Fully Dockerized development and production environments.
